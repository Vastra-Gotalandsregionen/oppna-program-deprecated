Öppna program

This site is managed by Region Västra Götaland (VGR) in Sweden. VGR’s assignment is to contribute to a thriving Västra Götaland which is a good place to live and work. Region Västra Götaland is governed by democratically elected politicians and with just over 50,000 employees is one of Sweden’s biggest employers.

The purpose of this site is to handle development projects using open source software and techniques. Therefore we gladly share what we develop e.g. to other similar institutions.

Discussions regarding the projects on this site is managed through this discussion group: http://groups.google.com/group/oppna-program-dev

open source community.
