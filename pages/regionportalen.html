Regionportalen

Regionportalen is a portal solution for the employees of VGR. Regionportalen is based on the portal server Liferay and other products within the

open source community.

  • Account activation - Portlet for inviting new users

  • Account management - Portlet for password handling

  • Alfresco Client - Portlet for Alfresco share integration with Liferay

  • Bookmarks - Portlet for adding personal bookmarks

  • Calender notfication - Portlet for Calender integration from several source e.g. Lotus Notes.

  • Commons - Common functions and utilities

  • CS-Iframe - Portlet enabling Simplified Single Sign-On, Credential Store Iframe.

  • Delegation service - Delegation service

  • DAO-framework - Framework for generic data access layer

  • exchange-integration - Integration with Microsoft Exchange

  • iFeed - Portlet for administration of ATOM based document flow subscriptions.

  • Liferay-user-associations - Liferay hooks to associate users to groups and update login attributes

  • Navigation - Portlets for simplified navigation within Regionportalen

  • Notification - Portlet and webservice. Liferay message bus notification adaptor

  • Piwik tracking codes - Portlet to handle web analytics codes for http://piwik.org piwik](

  • Portalramverk - GUI prototype for Regionportalen and Guidelines for user interface

  • Regionportalen custom-jsp-hook - Hook for custom-jsp

  • Regionportalen layout - Layout for Regionportalen

  • Regionportalen layout 2 - Layout 2 for Regionportalen

  • Regionportalen layout 3 - Layout 3 for Regionportalen

  • Regionportalen PDL - Adapt your health care application for PDL (patient data law)

  • Regionportalen theme - Theme for Regionportalen

  • Regionportalen theme 2 - Theme 2 for Regionportalen

  • Regionportalen theme 3 - Theme 3 for Regionportalen

  • RSS client - Portlet for displaying several RSS flows.

  • Signing Service - System for digital signature e.g. of documents

  • Tyck till - Portlet for user feedback handling

  • Tandvård beställning av intyg - Portlet for ordering dental certificates

  • User wizard - Wizard

  • Web Content Management - Data Structures for Regionportalen
