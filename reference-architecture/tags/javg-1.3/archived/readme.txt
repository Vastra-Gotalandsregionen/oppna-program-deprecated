Archived versions of JAVG
===========================
The old version of JAVG (before Q3 2008) have been deprecated and archived here
only for reference to the source.

The reasons for deprecation were:
- change of Maven groupId to have a proper "se.vgr.javg"-prefix
- restructuring of the code to be more easily digested by learners

The previous two javg-util and javg-examples Subversion repositories were also
merged to a common "javg" svn-repo.
