Referensapplikation JAVG

Starta applikation (med länk)

Detta är en exempel-applikation som är byggd enligt principerna för referensarkitekturen, med följande teknologier:

  • Web-ramverk: JSF (basen för hanteringen av web-requests), Facelets (vy-teknik)
  • Flödeskoordinering: Spring Web-flow (kontroll-lager pä flödesnivå)
  • Integrationsramverk för Lös koppling mellan komponenter: Spring Core (s.k. beroende-injicering)
  • Transaktionshantering: Spring Transaction, Spring AOP
  • Web-services: JAX-WS med Apache CXF
  • JAR-laddning av web-resurser: JAVG Support
